## Welcome

The `{{api_key}}` is located in your account on [Bad To Good Emails](https://badtogoodemails.com) under API Management API Keys.

All API endpoints are authenticated using the API key via QueryString parameter (HTTP GET) or the X-API-Key header (HTTP POST).

Please view the API EndPoints for Documetation on how to use each.
